# Responsive Nature Card

A responsive card component built using semantic HTML5 and CSS. This project features a clean and modern card layout with an image, category tags, descriptive content, and a call-to-action button.

## Features

* Semantic HTML5 structure
* Responsive design for different screen sizes
* Clean and modern UI
* Google Fonts integration
* Hover effects
* Mobile-friendly layout

## Technologies Used

* HTML5
* CSS3
* Google Fonts

```text
├── index.html
├── style.css
├── Nature.jpg

```
